그리드에 출력된 숫자는 formatString 에 의해 포매팅되어 출력된 모습입니다.

예로 첫 셀의 값인 "29,256" 의 실제 값은 29255.751 입니다. 이 실제값을 보기 좋게 "#,##0" 으로 포매팅하였습니다.

기본적으로 엑셀로 내보내기 할 때 "29,256" 은 그대로 "29,256" 으로 내보내기 됩니다.

그러나 엑셀 내보내기 할 때 29255.751 로 내보내기 할 수 있습니다.

xlsxNumeric을 true 로 설정하면 엑셀에서 더블클릭 시 보이는 값(또는 엑셀의 수식창의 값)은 29255.751 로 나타나게 됩니다.

참고 : xlsxNumeric 은 formatString 이외 prefix, postfix, labelFunction 등으로 포매팅한 경우 무시됩니다.(formatString 설정 시에만 사용 가능함)

또한 rounding 속성으로 소수점을 반올림처리 할 때만 유효합니다.

엑셀로 내보내기

var columnLayout = [{
	...
}, {
	dataField: "p1801",
	headerText: "`18 1월",
	dataType : "numeric",
	formatString : "#,##0",
	xlsxNumeric : true // 엑셀에 실제 숫자값으로 내보내기 함.
