푸터(footer) 의 가로 병합은 단순히 푸터 레이아웃 설정 시 colSpan 의 개수를 지정하십시오.

// 푸터 레이아웃 설정
var footerLayout = [ {
	dataField : "p1201",
	positionField : "p1201",
	operation : "SUM",
	colSpan : 3, // 자신을 포함하여 3개의 푸터를 가로 병합함.

위와 같이 colSpan 을 3으로 설정하면 포지션 칼럼(positionField)에 해당되는 "p1201"를 기준으로 3개의 푸터 셀을 병합하여 출력합니다.
